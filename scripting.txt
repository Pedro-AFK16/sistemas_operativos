###### find ######
find . -> lista de forma recursiva todos los archivos ocultos o no ocultos
-type f (a. regulares)
-printf -> le da formato 
	%p path
	%s size
	%t time modification
	%f file name
	\n salto de linea va a ultimo para que queden listados
	%m permisos
	%u user propietary

	(todo esto entre "%p  %t")
	ejemplo: find . -printf "%s  %u  \n" | column -t
-name (nombre del archivo)
-readable, -writable, -executable : filtrar por estos tipos
-size 150c -> la c al ultimo aclara que son bytes
-user
-group

##### file #####

##### tr #####
para sustituciones o borrado de caracteres especificos

##### sed #####
sustituir caracteres o palabras por otros caracteres o palabras
ejemplo1: cat pepe.txt | head -n 1 | sed "s/pedro/pedrito/g"
ejemplo2: sed "s/^ *//" -> borra espacios continuos
EXPLICACION DEL EJEMPLO1:
Del contenido elige la primera linea, y de ahi sustituye cualquier palabra "pedro" a "pedrito"
La "/g" especifica que sea para cada concidencia, si no pongo "/g" seria para la primer concidencia

##### expresiones regulares ##### 
"^r" -> los que empiezan por r
"^r$pepe" -> empieze con r y que termine con pepe
"^s.*\.sh$" -> empieze con s, ... , y termine con .sh
